# EUI 预设 Key 值列表

> 以下列出各预设 Key 类型当前所有可用值（`IsOpen=True`）。
> 预设值由配置表驱动，可能随版本更新变化，以实际数据表为准。

---

## ImageKey — 图片预设

数据来源：`data/client/ui_editor_img_style.py`

图片预设数量较多（300+），按分类组织，不在此逐一列举。  
包含：纯色背景、渐变背景、边框、图标、头像框、自定义图片等。  
特殊值：`-1` 表示透明/无图片。

---
