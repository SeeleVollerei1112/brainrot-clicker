# EUI 图片预设参考

用于 `GameAPI.create_eui_image_at_position()` 的 `preset_id` 参数。

## 特殊预设值

| 预设ID | 说明 | 用途 |
|--------|------|------|
| `-1` | 透明/无图片 | 创建透明容器，仅用于布局但不显示任何图像 |

**使用示例**：
```lua
-- 创建透明布局容器
local container = GameAPI.create_eui_image_at_position(
    -1,  -- 透明容器
    canvas,
    tf(960), tf(1000),
    tf(900), tf(80),
    "transparent_container"
)
```

## 使用示例

```lua
local tf = math.tofixed

-- 创建弹窗背景
local popup = GameAPI.create_eui_image_at_position(
    16347,  -- 蛋仔确认弹窗
    canvas, tf(960), tf(540), tf(740), tf(290),
    "popup_bg"
)

-- 创建物品格
local slot = GameAPI.create_eui_image_at_position(
    11084,  -- 圆形底图
    canvas, tf(100), tf(100), tf(80), tf(80),
    "slot_0"
)

-- 创建货币图标
local coin = GameAPI.create_eui_image_at_position(
    15056,  -- 乐园币
    canvas, tf(50), tf(50), tf(40), tf(40),
    "coin_icon"
)
```
